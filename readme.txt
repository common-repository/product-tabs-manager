== Description ==
[**Product Tabs Manager**](https://pluginever.com/plugins/wc-tab-manager/) is a powerful WordPress plugin for WooCommerce that allows you to create and manage custom product tabs with ease. With unlimited tabs, you can set global tabs across all products or assign unique ones to individual products. You can also rename WooCommerce's default tabs and rearrange custom tabs effortlessly. The plugin supports shortcodes, videos, image galleries, and HTML for dynamic content within tabs.

The [**Pro Version**](https://pluginever.com/plugins/wc-tab-manager/) unlocks advanced features such as category-specific tabs, Q&A-style tabs, product-type-specific tabs, and tab icons. It also allows you to exclude certain categories or products from showing specific tabs, disable default tabs, and includes a drag-and-drop sorting interface for easy tab management. Additionally, you can apply custom CSS for more control over the appearance and functionality.

== Installation ==
###Automatic Installation From WordPress Dashboard

1. Login to your admin panel
2. Navigate to Plugins â†’ Add New
3. Search **Product Tabs Manager By PluginEver**
4. Click install and activate respectively.

###Manual Installation From WordPress Dashboard

If your server is not connected to the Internet, then you can use this method-

1. Download the plugin by clicking on the red button above. A ZIP file will be downloaded.
2. Login to your site(s) admin panel and navigate to Plugins -> Add New -> Upload.
3. Click Choose file, select the plugin file, and click install

###Install Using FTP

If you are unable to use any of the methods due to internet connectivity and file permission issues, then you can use this method-

1. Download the plugin by clicking on the red button above. A ZIP file will be downloaded.
2. Unzip the file.
3. Launch your favorite FTP client. Such as FileZilla, FireFTP, CyberDuck etc. If you are a more advanced user, you can use SSH too.
4. Upload the folder to wp-content/plugins/
5. Log in to your WordPress dashboard.
6. Navigate to Plugins -> Installed
7. Activate the plugin

== Frequently Asked Questions ==
= What is the Product Tabs Manager plugin? =
The Product Tabs Manager plugin allows you to create, manage, and customize tabs on your product pages. It provides flexibility to display additional information, media, and custom content in organized tabs.

= Is the Product Tabs Manager plugin compatible with all WordPress themes? =
The plugin is designed to work with most WordPress themes. However, compatibility may vary depending on your theme's structure. If you encounter issues, please refer to the plugin's documentation or contact support.

= How do I exclude tabs for specific user types? =
In the Product Tabs settings go to **Exclude Tabs from User**, select the user types from which you want to exclude the tab. This will hide the tab for the selected user types.

=Is it possible to rename WooCommerceâ€™s default tabs? =
Yes, you can rename WooCommerceâ€™s default tabs using the Product Tabs Manager plugin. You can customize the tab names to suit your requirements and provide more relevant information to customers.

= Can I add icons to product tabs? =
The Pro version of the plugin allows you to add icons to product tabs. You can choose from a variety of icons to enhance the appearance of your tabs.

= How does the drag-and-drop sorting feature work? =
The drag-and-drop sorting feature allows you to rearrange the order of tabs by dragging and dropping them into your preferred sequence. You can easily customize the tab order to suit your requirements.

= Does the plugin support custom CSS? =
Yes, the plugin allows you to add custom CSS to style your tabs. You can add custom CSS in the plugin settings to customize the appearance of your tabs.
